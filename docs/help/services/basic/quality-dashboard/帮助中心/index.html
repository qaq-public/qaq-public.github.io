帮助中心

平台说明​

月报平台旨在帮助项目组快速汇总项目月度信息, 并对数据进行留档存储。 月报里的数据大致分为两个模块：

  • 每个月基本不变的信息, 如项目基本信息，项目成员，项目发布计划，项目里程碑等等，所以这部分信息配置在后台管理模块，新建月报时会自动把数据从后台管理同步到月报系统。

  • 每个月都会不同的信息，如产品风险，工作强度，人员变化，项目预算等需要创建月报时手动录入

配置项目信息​

后台管理会维护项目的基本信息和项目成员信息，维护此系统需要项目管理员权限(目前管理员一般为各项目主QA，可以联系添加)。

  • 进入后台管理 点击需要配置的项目
