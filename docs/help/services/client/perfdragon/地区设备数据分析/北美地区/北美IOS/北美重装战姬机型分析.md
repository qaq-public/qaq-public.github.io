# 北美重装战姬机型分析

## 数据来源

本数据取自北美地区iOS注册用户使用机型

## 游戏上线时间

上线时间：2019年11月21日

## 机型分布

**A13**及**A13以上**的机型用户占总机型的30%以上

**A12**及**A12以上**的机型用户占总机型的60%以上

**A10**及**A10以上**的机型用户占总机型的90%以上

| CPU | 品牌数量 | 品牌占比 |

| A14 | 7654     | 21.48%   |
| A13 | 9772     | 27.42%   |

| A11 | 4317     | 12.12%   |
| A10 | 2846     | 7.99%    |

## 内存分布
