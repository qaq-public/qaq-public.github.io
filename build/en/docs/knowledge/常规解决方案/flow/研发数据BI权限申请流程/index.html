研发数据BI权限申请流程

研发数据BI权限申请流程​

  1. OA申请

    在OA系统的[应用中心]-[游戏业务]内选择[游戏数据权限及服务申请]

    Preview

  2. 选择需要申请游戏

    Preview

    勾选研发数据

  3. 申请理由

需要补充：【如第一次开通 原无对于数据账号，需要申请数据账号： 企业微信名称（例如:chenweibo）】

游戏地区：【游戏发布地区 例如：TW、JP 】

项目数据源表：【 联系 莫得 获取对应的数据源表】
例如如下 stage_rd_sld_event_src_i_d （事件表） stage_rd_sld_user_snapshot_src_i_d （快照表）

申请原因：自研游戏数据埋点测试
