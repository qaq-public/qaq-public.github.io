生成Access tokens

1. 登录gitlab，点击右上角个人账户-> Settings​

Preview

2. 点击左边栏Access Tokens​

Preview

3. 输入token名称，勾选权限，点击绿色按钮添加（注意截至日期 "Expires at" 可不填）​

Preview

4. 创建成功后，显示token信息，复制到本地（注意：关闭此页面后Token码将再也看不见，务必复制）​

  • 1. 登录gitlab，点击右上角个人账户-> Settings
